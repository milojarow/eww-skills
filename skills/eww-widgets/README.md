# eww-widgets Skill

Widget selection guide, universal properties, magic variables, and deep reference
for every eww built-in widget.

## Files

| File | Contents |
|---|---|
| `SKILL.md` | Start here. Widget selection table, universal properties, magic variable quick reference, common patterns, critical callouts, best practices. |
| `ALIGNMENT.md` | Behavioral model for `halign`, `valign`, `hexpand`, `vexpand` — values, mental model, tug-of-war, common mistakes. |
| `CONTAINERS.md` | Layout and container widgets: `box`, `centerbox`, `overlay`, `scroll`, `stack`, `eventbox` |
| `INTERACTIVE.md` | User-input widgets: `button`, `checkbox`, `input`, `scale`, `combo-box-text`, `color-button`, `color-chooser` |
| `DISPLAY.md` | Output widgets: `label`, `image`, `progress`, `circular-progress`, `graph`, `systray`, `revealer`, `expander`, `transform`, `literal`, `tooltip`, `calendar` |
| `MAGIC_VARS.md` | Complete EWW_* reference with exact JSON structures, field names, units, and access patterns |
